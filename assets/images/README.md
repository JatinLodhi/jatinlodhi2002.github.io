# Images Directory

This directory contains image assets for the DevOps portfolio.

## Recommended Image Types

### Profile & Hero Images
- **profile-photo.jpg/png** - Professional headshot (500x500px recommended)
- **hero-background.jpg** - Hero section background (1920x1080px)

### Project Screenshots
- **project-[name]-screenshot.png** - Project showcase images
- **project-[name]-architecture.png** - Architecture diagrams

### Technology Logos
- **tech-[name]-logo.svg/png** - Technology stack logos (100x100px)

### Blog Assets
- **blog-[title]-featured.jpg** - Blog post featured images (800x450px)

## Optimization Guidelines

- **Format:** Use WebP for web images, PNG for transparency, JPEG for photos
- **Size:** Optimize images before upload (use tools like TinyPNG, ImageOptim)
- **Max Size:** Keep individual files under 500KB
- **Dimensions:** Use appropriate dimensions for use case
